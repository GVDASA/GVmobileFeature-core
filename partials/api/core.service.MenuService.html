MenuService
service in module core

Description

Realiza o gerenciamento do menu do aplicativo.

Methods

    Altera o item do menu para que seja exibido no desktop ou não.

    Parameters
    ParamTypeDetails
    menuobject

    Menu.

    shouldAddboolean

    Indica se deve ou não adicionar o item ao desktop.

  • getMenus()

    Carrega os menus que o usuário possui permissão.

  • resetMenu()

    Volta o menu ao estado original.

  • saveOnStorage()

    Salva as configurações do menu no storage do dispositivo.
