MenuProvider
service in module core

Description

Configura o serviço de menu.

Methods

  • addMenuItem(value)

    Adiciona um item ao menu.

    Parameters
    ParamTypeDetails
    valueobject

    Objeto de configuração do item do menu.

    value.namestring

    Nome que será exibido no item de menu.

    value.modulestring

    Nome do módulo a qual este item pertence.

    value.urlstring

    Url referente ao item de menu.

    value.iconstring

    Ícone que será exibido no menu.

    value.featurestring

    Nome da feature atrelada a este item de menu.

    value.desktop
    (optional)
    		boolean

    Indica se irá ser exibido no desktop ou não.

    value.order
    (optional)
    		number

    Ordem do item de menu no menu.

    value.cfgobject

    Configuração que será passada ao objeto $state do ui.router.
